# Employee Data Entry System

This project is a simple JSP + Servlet + JDBC web application.

## Features
- Employee data entry form
- Department and designation loaded from database
- Data stored using JDBC
- MySQL database integration
- Report generation using JOIN query

## Technologies Used
- Java
- JSP
- Servlets
- JDBC
- MySQL
- Apache Tomcat
